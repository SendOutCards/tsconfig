# tsconfig

Base tsconfig setttings

> Shared [TypeScript config](https://www.typescriptlang.org/docs/handbook/tsconfig-json.html) for SendOutCards projects

## Install

```
$ yarn add -D @sendoutcards/tsconfig
```

## Usage

`tsconfig.json`

```json
{
  "extends": "@sendoutcards/tsconfig",
  "compilerOptions": {
    ...
  }
}
```
